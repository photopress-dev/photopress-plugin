PhotoPress is a plugin that adds image/photo related features to WordPress.

== Description ==

This plugin allows you to import the XMP, EXIF, and IPTC for uploaded images, store that meta-data in specialized image taxonomies, and then display galleries based on taxonomy terms. It is also possible to display a random gallery of images.

= Features include: =

* import the XMP, EXIF, and IPTC for uploaded images
* Display image meta-data
* Create galleries based on image taxonomy terms
* Create a random gallery of images based on a taxonomy. Just add query="random" to the core gallery shortcode.

= The PhotoPress Plugin Family =

Most PhotoPress features are implemented as seperate extension plugins:

* [PhotoPress Image Taxonomies](http://wordpress.org/plugins/photo-tools-image-taxonomies/)
* [PhotoPress Gallery](http://wordpress.org/plugins/photopress-gallery/)
* [PhotoPress Paypal Shopping Cart](http://wordpress.org/plugins/photopress-paypal-shopping-cart/)
* [PhotoPress Masonry Gallery](http://wordpress.org/plugins/photopress-masonry-gallery/)

=== What happened to the pre 1.0.0 vesion of the plugin? ===

Version 1.0.0 is a complete rewrite and re-imagining of this plugin. Pre 1.0.0 versions of this plugin contained proprietary features that are now part now part of WordPress Core or no longer compatible with the latest versions of WordPress.
